# Simple AngularJS Project

A minimalist AngularJS application demonstrating basic concepts.

## Features

- Add items to a list
- Remove items from the list
- Display item count
- Simple, clean UI

## Files

- `index.html` - Main HTML file with styling
- `app.js` - AngularJS controller and logic
- `README.md` - This file

## How to Run

1. Open `index.html` in a web browser
2. Type an item name in the input field
3. Click "Add Item" or press Enter
4. Click "Remove" to delete items

## Technologies Used

- AngularJS 1.6.9
- HTML5
- CSS3

## Structure

```
myApp (AngularJS Module)
└── MainCtrl (Controller)
    ├── items (Array)
    ├── newItem (String)
    ├── addItem() (Function)
    └── removeItem() (Function)
```

Enjoy! 🎉
